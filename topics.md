# Встроенные функции

* `print()`
* `input()`
* `min()`
* `max()`
* `sum()`
* `len()`
* `str()`
* `int()`
* `float()`
* `list()`
* `bool()`
* `range()`

Пример использования функции `list()`:

```python
word = "слон"
lst = list(word)
print(lst)
```
Результат выполнения программы:

```
['c', 'л', 'о', 'н']
```
